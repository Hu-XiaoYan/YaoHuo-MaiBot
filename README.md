# 尧火MaiMai-Bot
### 使用方法
1.先确保已经安装以下Python第三方库:pillow,sqlite3  
2.前往ffmpeg官网下载最新软件包,并配置系统变量,重启计算机或服务器  
3.将plugins和src的内容粘贴至你的机器人内,并更改Public.py内的全局文件函数,将"yaohuo_bot"替换为你机器人plugin文件夹上一层文件夹的名字  
4.下载资源文件,解压至src文件夹中即可启动
